values relative to the top left corner and using the <code>align</code> and <code>verticalAlign</code> options. The <code>align</code> option can have the values <code>'left'</code>, <code>'right'</code> and <code>'center'</code>. The <code>verticalAlign</code> option can have the values <code>'top'</code>, <code>'bottom'</code> and <code>'middle'</code>.</p>\n<h3>Chart</h3>\n<p>The chart's position in the HTML page is defined by the position of the container div. Height and width is set either by setting a height and width of the container div, or by setting the <a href=\"https://api.highcharts.com/highcharts/chart.height\" target=\"_blank\">chart.height</a> and <a href=\"https://api.highcharts.com/highcharts/chart.width\" target=\"_blank\">chart.width</a> Highcharts options.</p>\n<p>The <em>plot area</em> is the square area within the axes, where the series are drawn. There are two sets of options that decide where the plot area is placed within the chart. The <em>margin</em> options (marginLeft, marginTop etc) give the margin to the actual plot area and does not apply to other elements on the chart (labels, legend etc). The <em>spacing</em> options (spacingLeft, spacingTop etc) decide the padding between the outer area and all elements inside a chart. In this mode, the chart is adjusted to make room for title and legend, axis labels etc. By default, the margins are null. Margins are only used in legacy charts as well as in situations where you want to make sure plot areas are aligned between multiple charts. Margins override their counterpart spacing options.</p>\n<p><img src=\"chartPosition.png\" alt=\"Illustration of margins, spacings, the plot area, and the plot border of a bar chart\"></p>\n<pre><code>chart: {\n        // Edit chart spacing\n        spacingBottom: 15,\n        spacingTop: 10,\n        spacingLeft: 10,\n        spacingRight: 10,\n\n        // Explicitly tell the width and height of a chart\n        width: null,\n        height: null\n}\n</code></pre>\n<h3>Credits</h3>\n<p>Credits can be moved around using the \"position\" option:</p>\n<pre><code>credits: {\n    position: {\n        align: 'left',\n        verticalAlign: 'bottom',\n        x: 10,\n        y: -10\n    }\n}\n</code></pre>\n<p>See <a href=\"https://api.highcharts.com/highcharts/credits\" target=\"_blank\">api.highcharts.com#credits</a> for full option set.</p>\n<h3>Axis labels</h3>\n<p>Axis labels can be placed relative to the plot area using the \"align\" option, \"x\" and \"y\" options.</p>\n<pre><code>labels: {\n    align: 'right',\n    x: -10,\n    y: 0\n}\n</code></pre>\n<p>They can also be rotated using the \"rotate\" option. This can be changed by the \"align\" and \"verticalAlign\" options. The legend can also be positioned anywhere in the chart area using the \"floating\" option.
