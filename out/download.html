下载 / 安装 Highcharts

您可以通过下面的方式获取及使用 Highcharts

直接使用 CDN 文件

无需下载，即可随时使用 Highcharts CDN 提供的文件资源。 

<scriptsrc="https://code.highcharts.com/10.3.3/highcharts.js"></script>

通过 NPM 安装

Highcharts 官方 NPM 包 提供了对 CommonJS 的支持，包含了 Highcharts Core、Stock、Maps 及 Gantt 所有相关的包。更多详情见 文档教程。

npm install highcharts --save

作为模块加载或创建您自己的构建版本

Highcharts 也可以从 CDN 服务中以 ECMAScript模块加载，也支持 AMD、UMD 模块或 TypeScript。您也可以按需求构建打包您自己的功能包。 详见 Github 参考中的说明文档

下载我们提供的资源包

压缩包包含我们提供的基础实例及相关的 JS 文件，下载后解压并打开 index.htm 即可看到我们提供的示例。
